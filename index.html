GeoPulse

Interactive World Explorer & Narrator

Built by Keerthana G Pillai

GeoPulse is a client-side web application I built to explore countries around the world. It uses the Fetch API to retrieve live country data, the Web Speech API to narrate country facts aloud, and the LocalStorage API to remember your favourites and preferences — all without any backend.

Features

  • Search & filter countries by name, capital, or region
  • Text-to-speech narration using the Web Speech API
  • Favorite shelf persisted with LocalStorage
  • Dark / Light theme with persistent preference
  • Quick link to OpenStreetMaps for every country
  • Sort by name or population

Fetch API
Web Speech API
LocalStorage API
